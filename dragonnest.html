Dragon Nest

Chibi Style action game


Dragon Nest is a Chibi style dungeon crawler MMO action RPG devolopped by Eyedentity game and released in 2010.
The publisher is Shanda Games.
Because the game was created such a long time ago, the graphics don't really match others MMO out there, but it still has its own style and artistic direction.
Also for the same reason, the game has much much contents, many dungeons, raids (nests), quests and others things like fishing, harvesting and cooking.

The game has many classes and they all have theyre own gameplay styles,and each of them has many cool skill, which is a pro and a con a the same time, because almost every skills are shortcut associated, there is only a very few command skills.

The story is pretty cool for an MMO, and you get to watch many awesome cinematics, other than that, its the same old "talking to NPC, go make a dungeon, return the quest and repeat", nothing exceptionnal here.
The leveling is not so fast, I have only been up to lvl 60 and each lvl already takes about 30 minutes (the cap lvl currently is 95)and each dungeon has many levels of difficutly but I'm not really sure what is the use, given that you can cruise trought the whole game doing it in normal difficulty.

The only thing that I found annoying in this game is that the monsters are kindof slippery, you almost always find yourself turning the camera around to be able to hit them, and sometimes you just can't hit them for some reason (like when they are knock down for exemple).

Overall, Dragon nest is a pretty fun game to play, and has its own graphic style, if you like chibi caracters and big monsters, this the game for you !

Classes

The game currently has more than 8 class which each has 4 different subclasses, when you reach lvl 15 you chose between 2 specializations, and when you reach lvl 45 you can chose again between 2 specializations.
Those 8 classes all have theyre kind of "evil counterpart" which has differents skills and only 1 specialization, but is a bit similar to the original class.
For exemple you have the "Warrior", who can specialize between Swordmaster and Mercenary, and you have the "Avenger" which is like the dark "warrior", who will upgrade into dark avenger later on.

Caracter customization

The caracter customization is not very deep, you can only change a few things, like the hair style and color, the eye color or the face, but not much to do here, if you are a fan of character customization, you will not find your happiness here.
